TransomJS is a rapid API development framework. Transom applications use metadata to configure pre-built and tested modules that allow developers to quickly piece together services that 'just work'. All transom applications are deployable and scalable on all the major hosting platforms. Using all standard Node libraries like Restify and Passport, you get all the benefits with minimal effort. TransomJS is based on plugins from the ground up. Everything is a plugin and can be modified or replaced. It is understood that every application has some special sauce that does not come from the framework!
Why should you use TransomJS:
  • Spend your time on your special sauce
  • Don't re-invent the wheel
  • Fewer bugs in less code, all the transomJS code is pre-tested!
  • Designed to be scalable
  • Security is a solved problem
  • ...

Transom Core

The transom core is the heart of the application. All other plugins are registered with the core.

MongoDB

The Transom mongoose plugin provides ready to use CRUD endpoints for all your data interactions backed by a Mongo database. Sophisticated querying, paging and sorting is all included, without writing any code.

Details
Security

Implement industry standard security. Choose which endpoints to secure. Apply fine grained security to your data!

Details
Email

Add email functionality to your application, just supply the configuration data.

Details
Custom Server Functions

Create your own end points and write your own code. Use all of the functionality provided by all the Transom plugins. Your own end points are easily secured, can send email, access the database and do anything a plugin can do.

Socket IO

Add secure Websockets for asynchronous communications. Have your server call you back when a process finishes. Synchronize your data across multiple clients in real time!
