Build your RESTful API faster

TransomJS allows developers to create beautiful REST-APIs with simple metadata. Turn an empty Restify server into a full-fledged API in minutes. Need something more? Go ahead, there are lots of built-in integration points and nothing in your way to write your own custom endpoints.

Learn more »

TransomJS is a rapid API development framework built on top of Restify. Transom uses metadata to configure pre-built and tested modules that allow developers to quickly piece together services that 'just work'. Transom applications are deployable anywhere you can run NodeJS. Using standard Node libraries like Restify, Mongoose and Passport, you get all the benefits with minimal effort. TransomJS is based on plugins from the ground up allowing new ones to be added and existing ones to get modified or replaced.

We understand that every application is going to have some special sauce that does not come from the framework and we've made sure it doesn't get in your way!
